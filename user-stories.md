User Stories in Agile

1\. Definition



A user story describes a feature from the perspective of an end-user.



2\. Standard Format



As a \[type of user], I want \[feature], so that \[business value].



3\. How to Write Effective User Stories



Good user stories are:



Independent



Negotiable



Valuable



Estimable



Small



Testable

(“INVEST” model)
